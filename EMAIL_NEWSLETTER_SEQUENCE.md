# Email Newsletter Sequence - 30 Days
**YPSMA Jombang | Yayasan Pendidikan Dan Sosial Maarif**

## Overview

- **Retention**: Keep donors engaged post-first donation
- **Education**: Explain impact and build trust
- **Upsell**: Convert one-time → recurring donors
- **Referral**: Encourage word-of-mouth sharing

**Test Variables**:
1. Subject lines (emoji vs plain)

## Quality Checklist

### Before Launch
- [ ] Proofread all 14 emails (grammar, spelling, facts)
- [ ] Test all WhatsApp links (pre-filled messages work?)
- [ ] Test all CTA buttons (correct landing pages?)
- [ ] Verify personalization variables ({{NAMA}} renders correctly?)
- [ ] Set up tracking pixels (open rate, click rate)
- [ ] Create fallback plain-text versions
- [ ] Test on mobile devices (80% open on mobile)
- [ ] Set up unsubscribe link (CAN-SPAM compliance)
- [ ] Create sender profile (from: ypsma@ypsma.org, reply-to: same)
- [ ] Warm up email domain (start with 50/day, scale to 500/day)

## Maintenance Schedule

### Monthly Updates
- Update donation totals (Rp X collected this month)
- Refresh testimonials (new donor quotes)
